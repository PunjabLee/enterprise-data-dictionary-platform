## 4. 禁止事项

- 禁止未确认技术栈就创建工程。
- 禁止 Service 直接调用 MyBatis Mapper。
- 禁止手工改数据库结构。
- 禁止把 token、密码、密钥提交到仓库。
- 禁止多个 agent 在同一分支并行写入。
- 禁止绕过 PR Steward 质量门禁直接合并到 `develop`。
- 禁止绕过 Product/Architecture Owner 决策机制扩大范围或改变架构路线。
