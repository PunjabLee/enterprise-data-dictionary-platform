# MVP 测试用例清单 v0.1

## 1. 测试目标

验证 MVP 是否满足资产目录、数据字典、业务术语、标准、导入、血缘、影响分析、审批、权限、安全和自动化治理的基础能力。

## 2. 功能测试

| 用例 | 场景 | 预期 |
| --- | --- | --- |
| TC-F-001 | 创建系统资产 | 资产进入草稿或待确认状态 |
| TC-F-002 | 发布资产 | 生成版本并进入已发布状态 |
| TC-F-003 | 维护字段字典 | 字段业务定义、标准、术语引用保存成功 |
| TC-F-004 | 创建业务术语 | 术语进入草稿状态 |
| TC-F-005 | 发布业务术语 | 术语可被字段和指标引用 |
| TC-F-006 | 创建数据标准 | 标准可提交评审 |
| TC-F-007 | 发布数据标准 | 标准可被字段引用 |

## 3. 导入测试

| 用例 | 场景 | 预期 |
| --- | --- | --- |
| TC-I-001 | 上传正确模板 | 生成导入批次 |
| TC-I-002 | 模板版本错误 | 返回模板版本错误 |
| TC-I-003 | 必填缺失 | 生成行级错误明细 |
| TC-I-004 | 重复编码 | 按策略拒绝、更新或待确认 |
| TC-I-005 | 部分成功 | 批次状态为 partial |

## 4. 血缘与影响分析测试

| 用例 | 场景 | 预期 |
| --- | --- | --- |

| TC-L-002 | 确认血缘 | 血缘可用于影响分析 |
| TC-L-003 | 从表发起影响分析 | 返回下游接口、任务、报表、指标 |
| TC-L-004 | 只看已确认血缘 | 待确认血缘不进入结果 |
