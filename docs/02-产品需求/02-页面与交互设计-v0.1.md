# 页面与交互设计 v0.1

## 1. 页面结构

| 一级菜单 | 二级页面 |
| --- | --- |
| 工作台 | 我的待办、我的关注、治理概览 |
| 资产目录 | 全部资产、资产详情、资产关系 |
| 数据字典 | 表字典、字段字典、差异确认 |
| 模型管理 | 概念模型、逻辑模型、物理模型、模型映射 |
| 标准术语 | 数据标准、业务术语、引用统计、整改清单 |
| 血缘分析 | 血缘图、影响分析、血缘确认 |
| 架构治理 | 架构评审、模型变更、标准发布、责任认领 |
| 采集导入 | 数据源、采集任务、导入批次、错误明细 |
| 系统管理 | 用户、角色、权限、审计日志、系统配置 |

## 2. 工作台

### 2.1 信息区块

- 我的待办：审批、确认、认责、整改。
- 我的关注：关注资产的变更提醒。
- 治理概览：资产数、核心表数、标准映射率、责任认领率、血缘覆盖率。
- 快捷入口：新增资产、导入模板、发起评审、发起影响分析。

### 2.2 交互要求

- 点击待办进入对应流程详情。
- 治理指标支持按业务域和系统过滤。
- 普通用户只展示查询入口和关注资产。

## 3. 资产目录

### 3.1 列表页

筛选条件：

- 资产类型。
- 业务域。
- 系统。
- 状态。
- 责任人。
- 标签。
- 是否核心资产。
- 是否敏感资产。

列表字段：

- 资产名称。
- 资产类型。
- 所属系统。
- 业务域。
- 状态。
- 责任人。
- 更新时间。

操作：

- 查看详情。
- 关注。
- 发起影响分析。
- 发起变更。

### 3.2 详情页

页签：

- 基础信息。
- 业务定义。
- 技术属性。
- 责任信息。
- 标准引用。
- 上下游关系。
- 血缘图。
- 版本记录。
- 审批记录。
- 审计日志。

## 4. 数据字典

### 4.1 表字典

支持按系统、数据库、Schema、状态筛选。

表详情展示：

- 表基本信息。
- 字段清单。
- 主键索引。
- 业务说明。
- Owner 和 Steward。
- 上下游血缘。

### 4.2 字段字典

字段列表展示：

- 字段名。
- 中文名。
- 类型。
- 是否主键。
- 是否必填。
- 是否关键字段。
- 是否敏感。
- 标准映射。
- 业务定义完整性。

字段编辑限制：

- 采集技术属性默认只读。
- 业务定义、标准映射、责任人可由授权角色维护。
- 已发布字段变更必须走审批或生成新版本。

## 5. 模型管理

### 5.1 概念模型

- 左侧业务域树。
- 中间概念实体列表或关系图。
- 右侧实体详情。

### 5.2 逻辑模型

- 实体列表。
- 属性列表。
- 实体关系。
- 标准引用。
- 映射物理对象。

### 5.3 物理模型

- 数据库、Schema、表、字段树。
- 采集结果与发布版本差异。
- 发起模型变更。

## 6. 血缘分析

### 6.1 血缘图

默认展示：

- 当前节点。
- 上游一层。
- 下游一层。

支持操作：

- 展开上游。
- 展开下游。
- 按资产类型过滤。
- 按确认状态过滤。
- 路径高亮。
- 切换表级和字段级。
- 导出影响清单。

### 6.2 影响分析

输入：

- 资产对象。
- 变更类型。
- 影响层级。
- 是否只看已确认血缘。

输出：

- 受影响资产。
- 影响路径。
- 责任人。
- 是否核心资产。
- 是否阻断变更。

## 7. 架构治理

### 7.1 评审单

评审单包括：

- 申请信息。
- 关联资产。
- 检查项。
- 影响分析。
- 附件。
- 审批意见。
- 审批记录。

### 7.2 检查项

检查项结果：

- 通过。
- 不通过。
- 不适用。
- 需补充。

不通过时必须填写问题说明、整改责任人和截止时间。

## 8. 采集导入

### 8.1 导入批次

展示：

- 批次号。
- 模板类型。
- 上传人。
- 上传时间。
- 总行数。
- 成功数。
- 失败数。
- 状态。

### 8.2 错误明细

展示：

- 行号。
- 字段名。
- 错误类型。
- 错误说明。
- 建议处理方式。

## 9. 系统管理

系统管理页面包括：

- 用户管理。
- 角色管理。
- 权限管理。
- 数据范围管理。
- 数据源管理。
- 审计日志。
- 参数配置。
