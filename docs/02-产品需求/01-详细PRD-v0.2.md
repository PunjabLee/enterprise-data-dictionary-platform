# 详细 PRD v0.2

## 1. 产品范围

首期产品目标是建设一个可运行的企业数据架构治理与元数据运营平台 MVP，支持资产导入、资产目录、数据字典、元模型配置、标准术语、基础血缘、影响分析、架构评审、模型变更、责任认领和权限审计。

## 2. MVP 模块

| 模块 | 首期必须支持 | 暂不支持 |
| --- | --- | --- |
| 首页工作台 | 待办、关注资产、治理指标、快捷入口 | 个性化布局配置 |
| 资产目录 | 搜索、筛选、详情、标签、责任人、状态 | 复杂资产评分 |
| 数据字典 | 表字段维护、业务定义、标准映射、批量导入 | 字段级全量自动血缘 |
| 模型管理 | 概念、逻辑、物理对象和映射 | 专业建模工具级绘图 |
| 标准术语 | 创建、发布、引用、整改清单 | 跨企业标准联邦 |
| 血缘管理 | 表级、报表级、关键字段级血缘 | 实时流全链路血缘 |
| 影响分析 | 从表、字段、接口、报表、指标发起 | 自动生成技术改造方案 |
| 流程审批 | 架构评审、模型变更、标准发布、血缘确认、认责 | 复杂 BPMN 自定义建模 |
| 采集导入 | Excel 导入、数据库元数据采集框架 | 全量 ETL/BI 工具适配 |
| 权限审计 | 角色、菜单、操作、数据范围、审计日志 | 细粒度属性级 ABAC |

| 角色 | 首期主要操作 |

| 企业架构师 | 查看架构资产，审批架构评审，检查影响范围 |
| 数据架构师 | 维护模型，审批模型变更，确认标准引用 |

## 4. 核心业务流程

### 4.1 资产导入与确认

1. 管理员或 Steward 上传 Excel 模板。
2. 系统校验字段完整性、枚举值、重复编码和引用对象。
3. 校验失败时生成错误明细。
4. 校验通过后生成导入批次。
5. 导入资产进入“待确认”状态。
6. 责任人确认后进入“已确认”或“已发布”状态。

### 4.2 系统建设架构评审

1. 系统 Owner 创建架构评审申请。
2. 填写系统、业务域、核心数据对象、接口、责任人、分类分级。
3. 系统自动检查标准引用、核心数据对象、责任人、影响范围。
4. 企业架构师、数据架构师、安全合规人员审批。
5. 通过后形成评审记录，并关联资产版本。

### 4.3 模型变更审批

1. 用户选择表、字段、逻辑实体、接口字段或指标发起变更。
2. 系统生成变更前后差异。
3. 系统发起影响分析。
4. 数据架构师和数据 Owner 审批。
5. 通过后发布新版本。

### 4.4 血缘确认

1. 系统通过导入、采集或人工维护生成血缘。
2. 血缘进入“待确认”状态。
3. 责任人确认或修正。
4. 已确认血缘进入正式影响分析范围。

## 5. 功能需求

### 5.1 资产目录

资产目录必须支持：

- 按关键字搜索资产名称、编码、中文名、业务定义、系统、责任人。
- 按业务域、系统、资产类型、状态、责任人、标签筛选。
- 查看资产详情，包括基础信息、属性、关系、责任、版本、血缘、审批记录。
- 关注资产，订阅变更。
- 标记核心资产、敏感资产、待治理资产。

### 5.2 数据字典

数据字典必须支持：

- 表和字段列表。
- 字段业务定义维护。
- 标准引用和术语引用。
- 敏感字段标记。
- 字段状态流转。
- 批量导入和导出。
- 与采集结果进行差异比对。

### 5.3 模型管理

模型管理必须支持：

- 概念实体、逻辑实体、逻辑属性、物理表字段管理。
- 概念到逻辑、逻辑到物理的映射。
- 模型版本。
- 模型变更审批。
- 模型对象详情和关系查看。

### 5.4 标准术语

标准术语必须支持：

- 数据标准分类。
- 业务术语维护。
- 标准发布流程。
- 标准引用统计。
- 未引用或不合规资产清单。

### 5.5 血缘与影响分析

血缘与影响分析必须支持：

- 维护上游、下游、关系类型、粒度、来源方式、确认状态。
- 图形化展示上下游关系。
- 从资产详情页发起影响分析。
- 生成受影响资产清单。
- 影响分析结果可作为审批附件。

### 5.6 权限审计

权限审计必须支持：

- 用户、角色、权限维护。
- 业务域和系统级数据范围控制。
- 敏感字段展示控制。
- 导入、导出、发布、审批、删除、权限变更审计。

## 6. 非功能需求

| 类型 | 要求 |
| --- | --- |
| 性能 | 常规搜索秒级响应，血缘图按需加载 |
| 安全 | 敏感字段默认受控，采集账号密码加密 |
| 可扩展 | 资产类型、属性和关系可配置扩展 |
| 可追溯 | 核心对象变更必须记录版本和审计 |
| 可运维 | 采集、导入、审批、索引任务有日志 |

## 7. 验收标准

- 能导入首批资产模板并生成资产目录。
- 能维护字段业务定义、标准映射和责任人。
- 能发起并完成架构评审。
- 能发起并完成模型变更审批。
- 能展示表级和关键字段级血缘。
- 能从核心表或报表发起影响分析。
- 能按角色和数据范围控制访问。
- 能查看关键操作审计日志。
