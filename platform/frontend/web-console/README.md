# Web Console

Vue 3 + TypeScript + Vite frontend skeleton for the enterprise data dictionary platform.

## Baseline

- Vue 3
- TypeScript
- Vite
- Ant Design Vue
- Vue Router

## Local Startup

```powershell
npm install
npm run dev
```

The development server runs on port `5173` and proxies `/api` to the backend on `http://localhost:8080`.

## Checks

```powershell
npm run type-check
npm run build
```
